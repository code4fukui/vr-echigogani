A WebXR demo built with Three.js that showcases 3D-scanned crab robot models and an animated VRM avatar.

## Demo

Experience the live demo here: **[https://code4fukui.github.io/vr-echigogani/](https://code4fukui.github.io/vr-echigogani/)**

The scene features two crab models and a radish avatar orbiting a central point. You can navigate using mouse orbit controls or view the scene in augmented reality on a WebXR-compatible device.

## Features

*   **VR/AR Ready:** Integrated with WebXR for an immersive AR experience.
*   **Dynamic Animation:** The crab models and VRM avatar circle the scene. The VRM avatar also features procedural bone-based animation for its limbs and neck.
*   **GLB & VRM Models:** Demonstrates loading and animating both `.glb` (crab robots) and `.vrm` (radish avatar) model formats.
*   **Performance Optimization:** The VRM avatar's skeleton and vertices are simplified at runtime to improve rendering performance.
