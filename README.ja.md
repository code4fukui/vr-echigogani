# vr-echigogani

[
![Demo](https://img.shields.io/badge/demo-online-green)
](https://code4fukui.github.io/vr-echigogani/)

Three.jsで構築されたWebXRデモです。3Dスキャンされたカニロボットのモデルと、アニメーションするVRMアバターを展示しています。

## デモ

ライブデモはこちらから体験できます: **[https://code4fukui.github.io/vr-echigogani/](https://code4fukui.github.io/vr-echigogani/)**

シーンでは、2つのカニのモデルと大根アバターが中心点を周回しています。マウスのオービットコントロールを使って視点を操作したり、WebXR対応デバイスを使用して拡張現実（AR）でシーンを表示したりすることができます。

## 特徴

*   **VR/AR対応:** WebXRを統合し、没入感のあるAR体験を提供します。
*   **ダイナミックなアニメーション:** カニのモデルとVRMアバターがシーンを周回します。また、VRMアバターの手足や首には、ボーンベースのプロシージャルアニメーションが実装されています。
*   **GLB & VRMモデル:** `.glb`（カニロボット）と`.vrm`（大根アバター）の両方のモデルフォーマットの読み込みとアニメーションを実演します。
*   **パフォーマンスの最適化:** VRMアバターのスケルトンと頂点を実行時に簡略化し、レンダリングパフォーマンスを向上させています。
